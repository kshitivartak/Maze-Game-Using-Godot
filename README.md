# Maze-Game-Using-Godot
The objective of this project was to design and develop a functional 3D maze game where the player must navigate through a series of corridors to find a specific exit. The project demonstrates the core pillars of 3D game development: Level Geometry, Player Control, Lighting, Audio, and Event Triggers.
